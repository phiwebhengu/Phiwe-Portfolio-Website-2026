A Very Simple Platformer

A complete 2D platformer game built from scratch in Unity — solo project.

Role
Game Designer & Developer
Tools
Unity, C#, Visual Studio
Platform
Itch.io (WebGL)

The Challenge

Design and build a complete, playable 2D platformer from scratch in Unity. The goal was to implement smooth player movement, responsive collision detection, and at least two distinct level environments — without relying on pre-built physics templates or asset store controller packages.

The Solution

I engineered a custom player controller using Unity's Rigidbody2D with manual velocity control. Rather than relying on Unity's default physics material for ground detection, I wrote a raycast-based ground-check, giving the player tight, responsive movement — a deliberate design choice to avoid the floaty feel common in physics-first approaches.

Technical Implementation

Core player movement script — handles horizontal input, raycast ground-check, and jump logic. 

// PlayerController.cs
using UnityEngine;

public class PlayerController : MonoBehaviour
{
    [SerializeField] private float moveSpeed = 6f;
    [SerializeField] private float jumpForce = 12f;
    [SerializeField] private LayerMask groundLayer;

    private Rigidbody2D rb;
    private bool isGrounded;

    void Start()
    {
        rb = GetComponent<Rigidbody2D>();
    }

    void Update()
    {
        // --- Horizontal movement ---
        float moveInput = Input.GetAxis("Horizontal");
        rb.velocity = new Vector2(moveInput * moveSpeed, rb.velocity.y);

        // --- Ground check via raycast ---
        isGrounded = Physics2D.Raycast(
            transform.position,
            Vector2.down,
            0.55f,
            groundLayer
        );

        // --- Jump ---
        if (Input.GetButtonDown("Jump") && isGrounded)
        {
            rb.velocity = new Vector2(rb.velocity.x, jumpForce);
        }
    }
}
